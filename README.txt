Este proyecto consta de diversos paquetes con diferentes clases 
el paquete numero uno es IMAGEN vienen las imagenes utilizadas en el proyecto
el paquete clasesauxiliares viene la clase redondeoDecimales la cual contiene un metodo que redodea los resultados
el paquete conversor vienen las clases que convierten un tipo de valor a otro 
el paquete interfaces viene la parte bonita y que ve el usuario, solo son interfaces que emplean las demas clases
